Semantic Ingredient-to-Emoji Matching

I built a vector-based similarity search to match ingredients to emojis using semantic embeddings instead of simple keyword matching.

Ingredient names are embedded using the gte-small model and stored in Supabase with the pgvector extension. Emoji assets live in Supabase Storage and are retrieved using cosine similarity search.
